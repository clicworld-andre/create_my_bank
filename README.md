# Clic Social Banking - Create My Bank

A modern landing page for the Clic Social Banking platform, enabling communities to create their own social banks.

## Overview

This website guides users through the journey of creating their own community bank using the Clic Wallet and Clic Social Banking platform. The process is designed to be simple and automated:

1. **Download Wallet** - Users download the Clic Wallet app
2. **Create Group** - Start a savings group within the wallet
3. **Invite Friends** - Build community with minimum 20 members
4. **Practice Banking** - Save together, access group loans
5. **Launch Social Bank** - Graduate to a full Social Bank

## Features

- Responsive design (mobile, tablet, desktop)

## Getting Started

Simply open `index.html` in a web browser or deploy to any static hosting service.

### Local Development

```bash
# Using Python
python -m http.server 8000

# Using Node.js
npx serve .

# Using PHP
php -S localhost:8000
```
